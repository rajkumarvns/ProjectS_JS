# Mini JavaScript Projects Collection

This repository contains beginner-friendly JavaScript projects created to improve problem-solving skills and strengthen HTML, CSS, and JavaScript fundamentals.

## 📌 Completed Projects

### 1. 👶 Age Calculator
Calculates a person's age based on the date of birth entered by the user.

**Features:**
- Accepts date of birth input
- Calculates years, months, and days
- Displays the exact age

---

### 2. 🧮 Calculator
A simple calculator that performs basic arithmetic operations.

**Features:**
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear functionality

---

### 3. ⚡ Electricity Calculator
Calculates electricity bills based on unit consumption.

**Features:**
- Enter consumed units
- Calculates total bill amount
- Displays final cost

---

### 4. 🎯 Guessing Number Game
A number guessing game where users try to guess the correct number.

**Features:**
- Random number generation
- User input validation
- Win/Lose messages
- Multiple attempts

---

### 5. 💰 Salary Calculator
Calculates net salary based on user inputs.

**Features:**
- Enter basic salary
- Calculate allowances and deductions
- Display final salary

---

### 6. ❌⭕ Tic Tac Toe
A two-player Tic Tac Toe game.

**Features:**
- 3×3 game board
- Player X and Player O turns
- Winner detection
- Reset game option

---

### 7. 💵 Tip Calculator
Calculates tips and total bill amount.

**Features:**
- Enter bill amount
- Select tip percentage
- Calculate total amount

---

### 8. ✅ To Do List
A task management application.

**Features:**
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed

---

### 9. ✈️ Travel Bill Calculator
Calculates travel expenses.

**Features:**
- Enter travel details
- Add transportation costs
- Add accommodation costs
- Calculate total expenses

---
